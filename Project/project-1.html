Note-Taking App

My Notes App is a full-featured note-taking web application built with React.js for the frontend and Node.js + MongoDB for the backend.

Live Link

Project Overview

This note-taking web application is built using React.js for the frontend and MongoDB for the backend, with a clean and user-friendly interface. Users can securely log in with token-based authentication, create colorful notes with randomly assigned background styles, edit and update content in real-time, and delete notes when they're no longer needed.

Notes are displayed in a responsive grid layout, and a built-in search feature allows users to quickly find specific content by typing keywords. The app also stores user sessions and data locally for a smoother experience, ensuring that user preferences are retained even after a refresh or logout. With its sleek design and essential features, the app is ideal for jotting down thoughts, to-do lists, and important ideas all in one place.
