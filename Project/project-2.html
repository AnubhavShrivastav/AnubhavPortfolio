Project Overview

This YouTube clone is a dynamic web application built using React.js and styled with Tailwind CSS, designed to provide a seamless video browsing and viewing experience. The app features a responsive layout with a fixed sidebar for easy navigation and a main content area that displays random videos fetched dynamically from the YouTube API. Users can explore a variety of video content, and clicking on any thumbnail redirects them to a dedicated video player page with detailed playback functionality.

The layout mirrors YouTube's familiar design, with a separate Navbar, Sidebar, and RandomVideos components, offering an organized and visually appealing interface. This clone provides a solid foundation for building more advanced features like search, user accounts, and playlists in the future.
